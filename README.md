# Tripleten web_project_around_express

## Proyecto 17: Alrededor de los EE.UU-express

Continuando con el desarrollo de aplicaciones web, este proyecto consiste en la creación de una plataforma social enfocada en compartir recuerdos y experiencias a través de imágenes. Para su desarrollo se utilizó React junto con Vite, permitiendo una aplicación moderna, rápida y dinámica.

## Características del proyecto

- HTML5 semántico
- Uso de Flexbox
- Posicionamiento de elementos
- Metodología BEM
- CSS3
- Diseño en Figma
- Git Bash
- GitHub
- GitHub Pages
- Manejo de ramas en Git
- Grid Layout
- JavaScript con JSX
- Integración de API
- React
- Vite

## Funcionalidad

La aplicación funciona como una galería interactiva donde se muestran diferentes lugares compartidos por miembros de la comunidad TripleTen. La página inicia con una sección de bienvenida y una breve descripción del proyecto.

Funciones implementadas:

- Visualización del perfil del usuario
- Sección de fotografías
- Actualización de información del perfil
- Eliminación de tarjetas con imágenes
- Sistema de likes operativo
- Publicación de nuevas imágenes
- Registro de usuarios
- Inicio de sesión
- Persistencia de sesión activa
- Cierre de sesión



## React + Vite

Este proyecto utiliza React en conjunto con Vite, proporcionando una configuración optimizada con soporte para Hot Module Replacement (HMR) y reglas de ESLint para mantener un código limpio y eficiente.

Actualmente existen dos plugins oficiales disponibles:

- @vitejs/plugin-react, que utiliza Babel para la actualización rápida.

- @vitejs/plugin-react-swc, que utiliza SWC para mejorar el rendimiento durante el desarrollo.
